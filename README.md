# imvigor210-survival-pipeline
This repository implements a reproducible machine-learning pipeline for overall survival prediction in anti–PD-L1–treated urothelial carcinoma patients using clinical covariates and RNA-seq data from the IMvigor210 trial.
## Data (Required)
This project requires the file:
- IMvigor210CoreBiologies.Rdata

For running the code, place it at:
./data/IMvigor210CoreBiologies.Rdata

Note: The dataset is not included in this repository (per submission instructions).
